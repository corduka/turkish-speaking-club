# 🇹🇷 Turkish Speaking Club

Master Turkish through immersive conversation. A modern web platform for boutique group lessons and personalized coaching.



## 🚀 Overview

**Turkish Speaking Club** is a specialized language learning platform designed to bridge the gap between "understanding" and "speaking." Built by a **professional language educator** and **Front-End Developer**, the project combines pedagogical expertise with a clean, high-performance user experience.

- **Live Site:** [Your-Link-Here.vercel.app]
- **Focus:** Boutique group sessions (Max 5 students) and Private 1:1 coaching.

## 🛠️ Tech Stack

- **Framework:** [React.js](https://reactjs.org/) (Vite)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Routing:** [React Router Dom](https://reactrouter.com/)

## ✨ Key Features

- **Responsive Design:** Optimized for mobile, tablet, and desktop with a custom-built hamburger menu system.
- **Dynamic Routing:** Seamless page transitions using React Router.
- **Micro-Animations:** Smooth scroll-reveal effects and interactive UI elements via Framer Motion.
- **The "Boutique" UI:** A minimalist aesthetic that emphasizes clarity and trust.
